## 🛠️ **Make It Yours**

- 1. Clone the repository and follow these simple commands to get started:
```bash
$ git clone https://github.com/syedmdfarhaneazam/ManagementApp
```
- 2. Install dependencies
```bash
$ npm install
```
- 3. Run the development server
```bash
$ npm run dev
